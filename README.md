# ⏰ Contador-Dev
Aplicativo desenvolvido na aula de Programação de aplicativo mobile

# 🤔 Como funciona?
A aplicação consiste em realizar um contador de quantas águas e cafés o programador ingeriu no dia, resetando após meia noite
